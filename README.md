# yalipay
Criação do app mobile YaliPay apresentando pela comunidade openSource Angolana.

## Tecnologias
2. Node.js
1. Ttpescript
2. Docker
3. Prisma


## Instalação

```bash
git clone https://github.com/Edson2001/yalipay.git
```

- Instalar dependências

```bash
npm install
```

- Executar o projecto

```bash
npm run dev
```

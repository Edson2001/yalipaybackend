# yalipaybackend

Criação do app mobile YaliPay apresentando pela comunidade openSource Angolana.

## Tecnologias
2. Node.js
1. Typescript
2. Docker
3. Prisma


## Instalação

```bash
git https://github.com/Edson2001/yalipaybackend.git
```

- Instalar dependências

```bash
npm install
```

- Executar o projecto

```bash
npm run dev
```
